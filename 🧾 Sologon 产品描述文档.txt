# 🧾 Sologon 产品描述文档

------

## 📌 产品概述

**Sologon** 是一款智能知识入库助手，帮助用户以极低的成本记录、管理和同步笔记、灵感、任务等碎片信息。它通过多端入口、智能标签推荐、自动分类和 GitHub 同步机制，构建个人轻量化的知识管理系统。

------

## 🎯 核心价值主张

> 给你的笔记，配一个“智能入库管理员”。

Sologon 让每一条灵感、记录、提醒都有去处、有结构、可查找、能归档。它不是笔记本，而是一位你身边的内容整理员。

------

## 🧩 核心功能

### 1. 多端快速记录

- Chrome 插件：浏览中快速记录
- Android App：移动随时捕捉
- 网页端：完整管理与搜索

### 2. 四大记录入口（交互友好）

- 🟢 **我干了**：记录已执行的行为/任务
- 🟡 **我想到了**：记录灵感、创意、点子
- 🔵 **提醒我**：带提醒的 TODO 或日程
- 🟣 **做记录**：自由内容笔记

### 3. 智能内容结构化

- **自动生成笔记标题**
- **语义识别推荐分类**（如“灵感”/“项目记录”/“资料收集”）
- **关键词提取生成标签**

### 4. 项目化管理系统

- 每条记录可关联到已有项目或创建新项目
- 项目中聚合所有相关笔记

### 5. GitHub 自动同步

- 所有记录自动整理到 GitHub 仓库中的 `note.md`
- 支持每日自动分文件归档
- 开关自定义，安全 Token 设置

## ⚙️ 技术结构简述

| 层级     | 技术选型建议 | 说明                             |
| -------- | ------------ | -------------------------------- |
| 前端     | React        | 支持 Chrome 插件 + 网页通用组件  |
| 移动端   | Flutter      | 最小 MVP 可用原生或 webview 实现 |
| 后端     | Python       | 快速原型，后期可换成微服务架构   |
| 存储     | SQLite       | 灵活文档结构适合笔记管理         |
| 同步模块 | GitHub API   | 用 Token 写入用户仓库            |
| 智能模块 | Gemini API   | 用于标题生成、分类识别、标签提取 |

## 🏁 最小可行产品（MVP）功能清单

| 模块         | 功能                  |
| ------------ | --------------------- |
| 快速记录面板 | 四种入口按钮、输入框  |
| 项目关联     | 可选择或创建项目      |
| 智能标签分类 | 自动标签 & 分类推荐   |
| 数据存储     | 本地+后端数据库       |
| GitHub 同步  | note.md 自动写入      |
| 前端原型     | Chrome 插件 或 网页端 |
